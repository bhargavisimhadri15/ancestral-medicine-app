## 🌿 Ancestral Medicine Plants Scanner

An intelligent plant scanning web application that helps users identify traditional medicinal plants and learn their ancestral health benefits instantly.

This project is built using modern web technologies and works as a Progressive Web App (PWA), allowing users to install it on mobile devices.

---

## 🚀 Features

* 📷 Plant scanning interface
* 🌱 Displays medicinal plant information
* 📱 Installable as mobile app (PWA)
* ⚡ Fast and responsive design
* 🌍 Works offline (Service Worker enabled)

---

## 🛠 Tech Stack
* HTML5
* CSS3
* JavaScript
* PWA (Service Worker + Manifest)
* Android Deployment Support

---

## 📦 Installation

```bash

python -m http.server 8000
```

App runs at:

```
http://localhost:8000
```
